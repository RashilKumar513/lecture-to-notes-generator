# Lecture to Notes Generator

A simple, static web-based tool that converts lecture content into summarized notes. This project includes HTML front-end files and is intended to be hosted as a static site.

## Features
- Paste or type lecture text and generate concise notes
- Clean, responsive HTML interface
- No backend required — runs entirely in the browser


## How to use
1. Open `index.html` in a web browser.
2. Paste your lecture text into the input area.
3. Click "Generate Notes" (or similar button) to see the summarized notes.

## Run locally
- Simply open `index.html` in your browser, or serve the directory with a simple static server:

  python -m http.server 8000

Then visit `http://localhost:8000`.
